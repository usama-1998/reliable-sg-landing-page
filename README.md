# Reliable SG - Instant Battery Replacement

Reliable SG provides instant on-site battery replacement services for cars in Singapore. We are committed to getting you back on the road quickly and safely.

## Features

- **24/7 Availability**: We are ready to help whenever you need us.
- **Instant Replacement**: fast on-site service at your location.
- **Professional Technicians**: Experienced and skilled staff.
- **Quality Batteries**: Top-tier batteries with warranty assurance.

## Tech Stack

This landing page is built with modern web technologies:

- **[Next.js](https://nextjs.org/)**: The React Framework for the Web.
- **[Tailwind CSS](https://tailwindcss.com/)**: Rapidly build modern websites without ever leaving your HTML.
- **[Lucide React](https://lucide.dev/)**: Beautiful & consistent icon toolkit.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
